# Paper PDFs

Put downloadable PDFs of your papers here, then link them from `publications.html`.
A commented template already sits next to each entry's DOI — just drop the file and
uncomment (or tell me and I'll wire it up).

Tip: most publishers (IEEE, ACM, Springer) let authors self-archive the **accepted
manuscript** (post-print). Posting that version here is the safest way to offer a direct
download for paywalled papers.
